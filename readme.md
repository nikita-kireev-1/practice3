# Второй этап
## Документация спроектированного языка ассемблера
### Список команд
#### 1. LOAD_CONST

Формат: {
"op": "LOAD_CONST",
"reg": 29,
"value": 27
}

"op" - название операции

"reg" - регистр отвечающий за адрес

"value" - значение загружаемой константы

#### 2. READ_MEM

Формат: {
"op": "READ_MEM",
"base_reg": 20,
"dst_reg": 16,
"offset": 390
}

"op" - название операции

"base_reg" - регистр содержащий адрес откуда считываем

"dst_reg" - регистр в который записываем

"offset" - целочисленное смещение для считывваемого регистра

#### 3. WRITE_MEM

Формат: {
"op": "WRITE_MEM",
"reg": 24,
"addr": 178
}

"op" - название операции

"reg" - регистр источника данных

"addr" - регистр куда записываем данные

#### 4. ADD

Формат: {
"op": "ADD",
"reg": 19,
"addr": 956
}

"op" - название операции

"reg" - регистр источника данных и туда же записываются данные

"addr" - адрес памяти откуда берётся второе слагаемое
